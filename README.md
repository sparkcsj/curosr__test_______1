# PDF 처리 웹앱

Flask 기반의 PDF 페이지 추출 및 파일 병합 웹 애플리케이션입니다.

## 기능

- **페이지 추출**: PDF 파일에서 원하는 페이지를 추출하여 새로운 PDF 파일로 저장
- **파일 병합**: 여러 개의 PDF 파일을 하나로 병합

## 설치

1. Python 3.8 이상이 설치되어 있어야 합니다.

2. 필요한 패키지를 설치합니다:
```bash
pip install -r requirements.txt
```

## 실행

애플리케이션을 실행합니다:
```bash
python app.py
```

브라우저에서 다음 주소로 접속하세요:
```
http://localhost:5000
```

## 사용 방법

### 페이지 추출

1. "페이지 추출" 섹션에서 PDF 파일을 선택합니다.
2. 추출할 페이지 번호를 입력합니다:
   - 개별 페이지: `1, 3, 5` 또는 `1 3 5`
   - 페이지 범위: `2-5` (2페이지부터 5페이지까지)
   - 혼합: `1, 3-5, 10` (1페이지, 3~5페이지, 10페이지)
3. "페이지 추출" 버튼을 클릭합니다.
4. 처리된 파일이 자동으로 다운로드됩니다.

### 파일 병합

1. "파일 병합" 섹션에서 여러 PDF 파일을 선택합니다.
   - Windows: `Ctrl` 키를 누른 상태로 파일 선택
   - Mac: `Cmd` 키를 누른 상태로 파일 선택
2. "파일 병합" 버튼을 클릭합니다.
3. 병합된 파일이 자동으로 다운로드됩니다.
   - 파일은 선택한 순서대로 병합됩니다.

## 제한 사항

- 최대 파일 크기: 50MB
- 지원 파일 형식: PDF만 가능

## 디렉토리 구조

```
PDFExtract/
├── app.py                 # Flask 메인 애플리케이션
├── templates/
│   └── index.html        # 메인 UI 페이지
├── static/
│   └── style.css         # 스타일시트
├── uploads/              # 업로드된 파일 임시 저장 (자동 생성)
├── downloads/            # 처리된 파일 저장 (자동 생성)
├── requirements.txt      # Python 의존성
└── README.md            # 사용 설명서
```

## 기술 스택

- **Flask**: Python 웹 프레임워크
- **pypdf**: PDF 파일 처리 라이브러리
- **Werkzeug**: 파일 업로드 처리

## 주의사항

- 업로드된 파일과 처리된 파일은 서버에 임시로 저장되며, 다운로드 후 자동으로 정리됩니다.
- 프로덕션 환경에서 사용하려면 추가 보안 설정이 필요할 수 있습니다.
